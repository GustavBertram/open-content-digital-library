# Open Content Digital Library

This is a collection of educational books, carefully selected to be *legally* redistributable. This means that these works may be freely copied and given away.

Licenses are typically noted on the filename, as are publication dates where they could be determined.

## STEM

The primary focus is on science, technology, engineering, and mathematics textbooks, including computer science and programming.

## Military Engineering Manuals

The collection also includes a number of public domain engineering training manuals created by the US military. These are immediately public domain when published because they were created by public funds, so the information should be relatively current.
